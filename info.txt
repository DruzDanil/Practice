Для роботи програм необхідно виконати наступні програми
python -m pip install requests
pip install Flask
pip install Flask-RESTful
Якщо виникнуть проблеми з запуском, будь-ласка перегляньте, чи всі бібліотеки підключені
Для початку роботи необхідно спочатку запустити файл Classes.py, потім працювати в файлі local.py
